Installation

Download the latest release:

github.com/stephband/details-toggle/releases

Include the JS and CSS files:

<link rel="stylesheet" href="./details-toggle.css" />
<script type="module" src="./details-toggle.js"></script>

<details-toggle>

Import <details-toggle> custom element. This registers the custom element and upgrades instances already in the DOM.

<script type="module" src="bolt/elements/details-toggle.js"></script>

Use it thus:

Attributes

A boolean attribute describing the state of the details-toggle.
